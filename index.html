Use if else when

  • Dealing with complex conditions with multiple branches

Use Ternary when

Functions in Javascript

Javascript has 2 ways of declaring functions, traditional notation and fat arrow notation

Use functions to wrap logic to make it reusable. They are great for avoiding mutations

Traditional Notation

function calculateSomething () { // enter code here }

Fat Arrow Notation

const calculateSomething = () => { // enter code here }

Function Syntax Specifics

  • The return statement in a function is the output value. it's not necessary but is a good idea to actually return something.
  • Function paremeterare the variables that you set up in the parentheses (). These are then referred to in the function. They make the function more reusable
  • Function argumentare the arguments in use. when you run the function you'll pass information into it that is then represented by the parameters
